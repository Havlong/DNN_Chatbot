# DNN_Chatbot
Simple DNN chatbot for Telegram

Language used:
- Python 3.9

Libraries used:
- `TeleBot`
- `TensorFlow Keras`
- `NLTK`

To start bot launch `python3 main.py` and it will start long-polling messages for Bot

Be sure to install required libs: ```python3 -m pip install tensorflow nltk telebot ```
